# Battleship

A browser-based implementation of the classic Battleship game. Play against another human or an intelligent computer opponent on two 10×10 grids.

## Features

* Play against a human or computer opponent
* Two 10×10 game boards
* Place ships horizontally before the game starts
* Fleet includes ships of lengths 5, 4, 3, 3, and 2
* Computer opponent targets adjacent cells after landing a hit
* Status panel keeps players informed throughout the game
* Victory and defeat music

## Built With

* HTML
* CSS
* JavaScript

## Usage

1. Choose a human or computer opponent.
2. Place your ships on the board.
3. Take turns attacking your opponent's grid.
4. Sink all enemy ships to win.
